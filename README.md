# 💬 Chat Room Management System

A console-based Chat Room Management System built using Python and Object-Oriented Programming (OOP).

## Features

- User Login System
- Multiple Chat Rooms
- Public Messaging
- Private Messaging
- Admin Controls
- Chat History
- Join & Leave Rooms

## Technologies Used

- Python
- Object-Oriented Programming (OOP)

## How to Run

```bash
python ChatRoom-System.py
```
